Supporting visual assets for Suns Summer League Pulse:

- `avc-logo-full-dark.svg`: Accelerated Velocity Consulting logo used in the header, footer, and OG preview.
- `phoenix-suns-logo.png`: Phoenix Suns mark used as a restrained subject seal in the OG preview.
- `vegas-courtside-v2.jpg`: Generated courtside editorial image used as the Issue 01 cover.
- `summer-league-courtside-bridges.jpg`: User-supplied courtside photo used for the Issue 03 and Issue 04 OG previews. Reuse permission was confirmed by the user; the embedded PHNX Suns watermark must remain visible.
- `khaman-maluach-game-02.png`: User-supplied game screenshot used for the Issue 04 OG preview. Reuse permission was confirmed by the user; the embedded caption and ESPN game context must remain visible.
- `koa-peat-game-03.jpg`: User-supplied Game 03 photo used for the Issue 05 OG preview.
- `khaman-maluach-three-double-doubles.jpg`: User-supplied Maluach graphic used for the Issue 06 OG preview.
- `suns-team-game04.jpg`, `koa-peat-game-03.jpg`, and `suns-spurs-game05.jpg`: User-supplied game photos used in the Issue 10 editorial OG montage.
- `issue-10-vegas-final-report-og.jpg`: 1200x630 edition-specific preview for the Vegas Final Report. Its editable source is `../og-preview-issue10.html`.

The root `../og-image.jpg` is synchronized to the current edition. Live metadata may point to that file or to an edition-specific image such as `issue-10-vegas-final-report-og.jpg`; either URL must be absolute. Visual source and usage notes should be checked before each issue is published.
